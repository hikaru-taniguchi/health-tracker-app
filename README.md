# 体調記録アプリ

## 概要
日々の体調や気分、飲酒の有無、しんどくなった要因・楽になった要因を記録するためのWebアプリです。  
体調の波を見える化し、自分の行動傾向を把握することを目的に作成しました。

## デモ
https://hikaru-taniguchi.github.io/health-tracker-app/

## 機能
- 日付の入力
- 体調点数の記録
- 前日・当日の飲酒の有無
- 気分メモの入力
- しんどくなった要因の記録
- 楽になった要因の記録
- 記録一覧への表示

## 使用技術
- HTML
- CSS
- JavaScript

## 工夫した点
- 1日の記録をオブジェクトとしてまとめ、配列で管理する形にした
- 毎日使いやすいように、入力項目を増やしすぎずシンプルにした
- スマホでも使いやすいようにレスポンシブ対応を入れた

## 今後の改善予定
